# Concept 06

## Goal
Dashboard-first story with Chart.js graphs for synthetic risk and mileage.

## Tech
- HTML
- CSS
- Vanilla JS
- Chart.js (CDN)

## Features
- Risk bar chart
- Mileage line chart
- Fake scores

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
