# Concept 04

## Goal
Generous whitespace, system typography, and soft section reveals.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- CSS smooth scroll
- IntersectionObserver fade-ins
- System / SF-like stack

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
