# Concept 17

## Goal
Apple-like tall page: panels shift with scroll for a product-story rhythm.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Parallax layers
- Pinned headline
- Section opacity choreography

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
