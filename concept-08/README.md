# Concept 08

## Goal
Vertical history spine with hover-expanded inspection nodes.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Animated timeline rail
- Hover detail drawers
- Keyboard-focusable rows

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
