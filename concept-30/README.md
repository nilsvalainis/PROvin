# Concept 30

## Goal
Each reload randomizes hero layout, palette, and density — generative UI sketch.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Random palette
- Shuffle section order
- Density mode

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
