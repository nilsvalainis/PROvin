# Concept 01

## Goal
Transparent glass UI with depth, fake VIN intelligence panel, and ambient light that follows the pointer.

## Tech
- HTML
- CSS (backdrop-filter)
- Vanilla JS

## Features
- Backdrop blur glass panels
- Animated hover lifts
- Fake VIN decode result card
- Mouse-driven radial highlight (CSS variables)

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
