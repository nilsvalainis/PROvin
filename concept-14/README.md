# Concept 14

## Goal
Single VIN field with instant validation feedback — nothing else competes for attention.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- 17-char VIN gate
- Live status chip
- No secondary chrome

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
