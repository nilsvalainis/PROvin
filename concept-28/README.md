# Concept 28

## Goal
No frameworks, tiny assets, readable single-purpose landing.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Sub-100KB total (excluding browser)
- No CDN dependencies
- Instant paint

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
