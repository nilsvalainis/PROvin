# Concept 19

## Goal
Strict black/white system with contrast through scale and spacing alone.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- No chroma accents
- Bold type ramp
- Grid rhythm

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
