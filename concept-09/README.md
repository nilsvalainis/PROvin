# Concept 09

## Goal
Theme toggle with smooth variable-driven transitions across surfaces.

## Tech
- HTML
- CSS custom properties
- Vanilla JS

## Features
- Animated theme toggle
- CSS variables for surfaces
- Persisted preference (localStorage)

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
