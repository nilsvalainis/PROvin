# Concept 26

## Goal
Leaflet map with a fake origin marker — swap tiles in production.

## Tech
- HTML
- CSS
- Vanilla JS
- Leaflet (CDN)

## Features
- OSM tiles
- Marker popup
- Responsive map height

## How to run
In development/production, open `/concept-demos/concept-26/` from the PROVIN site, or open `public/concept-demos/concept-26/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-26/
```
