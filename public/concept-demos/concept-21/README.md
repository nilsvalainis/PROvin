# Concept 21

## Goal
GSAP-powered micro-interactions on a dense control strip.

## Tech
- HTML
- CSS
- Vanilla JS
- GSAP (CDN)

## Features
- Staggered chip press
- Elastic hero nudge
- Reduced-motion guard

## How to run
In development/production, open `/concept-demos/concept-21/` from the PROVIN site, or open `public/concept-demos/concept-21/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-21/
```
