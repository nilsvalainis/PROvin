# Concept 02

## Goal
ChatGPT-style auto expert with typing animation and canned VIN insights.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Threaded chat UI
- Typing indicator
- Fake AI replies tied to VIN

## How to run
In development/production, open `/concept-demos/concept-02/` from the PROVIN site, or open `public/concept-demos/concept-02/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-02/
```
