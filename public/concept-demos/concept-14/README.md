# Concept 14

## Goal
Single VIN field with instant validation feedback — nothing else competes for attention.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- 17-char VIN gate
- Live status chip
- No secondary chrome

## How to run
In development/production, open `/concept-demos/concept-14/` from the PROVIN site, or open `public/concept-demos/concept-14/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-14/
```
