# Concept 28

## Goal
No frameworks, tiny assets, readable single-purpose landing.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Sub-100KB total (excluding browser)
- No CDN dependencies
- Instant paint

## How to run
In development/production, open `/concept-demos/concept-28/` from the PROVIN site, or open `public/concept-demos/concept-28/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-28/
```
