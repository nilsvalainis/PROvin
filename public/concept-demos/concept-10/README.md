# Concept 10

## Goal
Composable tiles with hover expansion for dense feature discovery.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Reusable card module
- Hover height expansion
- Accent rail

## How to run
In development/production, open `/concept-demos/concept-10/` from the PROVIN site, or open `public/concept-demos/concept-10/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-10/
```
