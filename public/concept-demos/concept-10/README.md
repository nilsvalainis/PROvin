# Concept 10

## Goal
Composable tiles with hover expansion for dense feature discovery.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Reusable card module
- Hover height expansion
- Accent rail

## How to run
Each page loads **Tailwind CSS** from the official CDN plus a small companion `styles.css` for bespoke visuals. Asset URLs are **root-relative** (`/concept-demos/concept-10/…`) so CSS/JS load even without a trailing slash on the folder URL.

In development/production, open `/concept-demos/concept-10/` from the PROVIN site. For CDN-heavy concepts (maps, charts, video), prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-10/
```
