# Concept 08

## Goal
Vertical history spine with hover-expanded inspection nodes.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Animated timeline rail
- Hover detail drawers
- Keyboard-focusable rows

## How to run
In development/production, open `/concept-demos/concept-08/` from the PROVIN site, or open `public/concept-demos/concept-08/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-08/
```
