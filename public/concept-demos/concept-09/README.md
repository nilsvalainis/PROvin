# Concept 09

## Goal
Theme toggle with smooth variable-driven transitions across surfaces.

## Tech
- HTML
- CSS custom properties
- Vanilla JS

## Features
- Animated theme toggle
- CSS variables for surfaces
- Persisted preference (localStorage)

## How to run
In development/production, open `/concept-demos/concept-09/` from the PROVIN site, or open `public/concept-demos/concept-09/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-09/
```
