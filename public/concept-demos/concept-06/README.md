# Concept 06

## Goal
Dashboard-first story with Chart.js graphs for synthetic risk and mileage.

## Tech
- HTML
- CSS
- Vanilla JS
- Chart.js (CDN)

## Features
- Risk bar chart
- Mileage line chart
- Fake scores

## How to run
In development/production, open `/concept-demos/concept-06/` from the PROVIN site, or open `public/concept-demos/concept-06/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-06/
```
