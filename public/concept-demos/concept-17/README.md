# Concept 17

## Goal
Apple-like tall page: panels shift with scroll for a product-story rhythm.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Parallax layers
- Pinned headline
- Section opacity choreography

## How to run
In development/production, open `/concept-demos/concept-17/` from the PROVIN site, or open `public/concept-demos/concept-17/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-17/
```
