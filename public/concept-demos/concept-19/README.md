# Concept 19

## Goal
Strict black/white system with contrast through scale and spacing alone.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- No chroma accents
- Bold type ramp
- Grid rhythm

## How to run
In development/production, open `/concept-demos/concept-19/` from the PROVIN site, or open `public/concept-demos/concept-19/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-19/
```
