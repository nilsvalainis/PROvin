# Concept 22

## Goal
Phone chrome with swipeable story cards (touch + mouse drag).

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Device frame
- Pointer swipe
- Snapping index

## How to run
In development/production, open `/concept-demos/concept-22/` from the PROVIN site, or open `public/concept-demos/concept-22/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-22/
```
