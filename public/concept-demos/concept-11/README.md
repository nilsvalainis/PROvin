# Concept 11

## Goal
Canvas-driven energy orb with pulsing gradients — no heavy WebGL stack.

## Tech
- HTML
- CSS
- Canvas 2D
- Vanilla JS

## Features
- Animated gradient orb
- Pointer parallax nudge
- Lightweight loop

## How to run
In development/production, open `/concept-demos/concept-11/` from the PROVIN site, or open `public/concept-demos/concept-11/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-11/
```
