## How to run
In development/production, open `/concept-demos/concept-07/` from the PROVIN site, or open `public/concept-demos/concept-07/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-07/
```
