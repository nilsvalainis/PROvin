# Concept 03

## Goal
High-contrast black/white layout with raw typography and zero friendly rounding.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- No border-radius
- Strict grid
- Oversized type scale

## How to run
In development/production, open `/concept-demos/concept-03/` from the PROVIN site, or open `public/concept-demos/concept-03/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-03/
```
