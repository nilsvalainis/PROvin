# Concept 13

## Goal
Asymmetric tile grid with mixed card spans like Apple marketing grids.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- CSS grid spans
- Soft glass tiles
- Live clock micro-widget

## How to run
In development/production, open `/concept-demos/concept-13/` from the PROVIN site, or open `public/concept-demos/concept-13/index.html` locally. For CDN-heavy concepts, prefer the deployed or `next dev` origin (not `file://`).

```bash
npm run dev
# then visit http://localhost:3000/concept-demos/concept-13/
```
