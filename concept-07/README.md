# Concept 07

## Goal
Sticky split layout: narrative on the left, vehicle canvas on the right.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Two-column split
- Sticky visual rail
- Scroll-tied caption swaps

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
