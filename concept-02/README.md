# Concept 02

## Goal
ChatGPT-style auto expert with typing animation and canned VIN insights.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Threaded chat UI
- Typing indicator
- Fake AI replies tied to VIN

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
