# Concept 10

## Goal
Composable tiles with hover expansion for dense feature discovery.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- Reusable card module
- Hover height expansion
- Accent rail

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
