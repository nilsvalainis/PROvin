# Concept 13

## Goal
Asymmetric tile grid with mixed card spans like Apple marketing grids.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- CSS grid spans
- Soft glass tiles
- Live clock micro-widget

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
