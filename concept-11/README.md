# Concept 11

## Goal
Canvas-driven energy orb with pulsing gradients — no heavy WebGL stack.

## Tech
- HTML
- CSS
- Canvas 2D
- Vanilla JS

## Features
- Animated gradient orb
- Pointer parallax nudge
- Lightweight loop

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
