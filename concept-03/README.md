# Concept 03

## Goal
High-contrast black/white layout with raw typography and zero friendly rounding.

## Tech
- HTML
- CSS
- Vanilla JS

## Features
- No border-radius
- Strict grid
- Oversized type scale

## How to run
Open `index.html` in a browser (double-click or via a static server). For concepts that load CDN assets (maps, charts, GSAP), use a simple static server so subresources load reliably:

```bash
npx serve .
```
